# 自动拆装箱与NPE

有代码：

```java
    boolean flag = true;
    Integer i = 0;
    int j = 1;
    int k = flag ? i : j;
```

编译，反编译：

```java
    boolean flag = true;
    Integer i = Integer.valueOf(0);
    int j = 1;
    int k = flag ? i.intValue() : j;//自动拆箱，若i == null，产生NPE。
    System.out.println(k);
```
